Automatic Email Scambaiting Robot

Hi everyone,

I’m currently building an automatic email scambaiting robot. It uses ChatGPT to generate realistic email replies, sends them via Gmail (SMTP/IMAP), and can read incoming responses. If anyone would like to safely test it, you can chat with it at susan.m.mills1954@gmail.com.

Why I made it:

I was inspired by scam-baiting YouTubers like Kitboga (Weston Amber), Scammer Payback, Pleasant Green, and others. I wanted to create an automated version that could respond realistically without being glued to my inbox all day.

What I’m working on now:

I have a working prototype of an automatic email responder. The next step is adding an autocaller, but VOIP services are expensive (~£10/month), so I’m holding off for now.

How to safely test:

  • Send harmless messages to the account to see how it responds.
  • Do not send personal information.
  • All responses are automated, so keep messages appropriate.

Technical details:

  • Built in Python with Flask for the dashboard.
  • Uses OpenAI’s API for generating realistic replies.
  • Logs all activity in a console/dashboard for review.

I’d love feedback, suggestions for features, or help testing new scenarios. Let me know if you want to see it handle different types of scam emails!
